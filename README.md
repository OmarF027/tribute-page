# Tribute Page to Ryuk

This is a tribute page dedicated to **Ryuk**, the Shinigami (god of death) from the **Death Note** manga and anime series.

## Description

Ryuk is a Shinigami who dropped his **Death Note** into the human world, altering the course of humanity forever.
